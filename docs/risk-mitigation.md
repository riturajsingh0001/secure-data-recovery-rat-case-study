# ⚠️ Risk Mitigation

## 📊 Risk vs Mitigation Table

| Risk              | Description               | Mitigation               |
| ----------------- | ------------------------- | ------------------------ |
| Malware Execution | Opening infected files    | No file execution policy |
| System Infection  | Malware spreading to host | Used Live OS             |
| Data Corruption   | Improper device removal   | Safe eject process       |
| Network Attack    | Malware communication     | Internet disabled        |
| USB Infection     | Malware transfer via USB  | No execution + post-scan |

---

## 🔒 Security Strategy

* Isolation (Live OS)
* Minimal interaction (copy only)
* No internet connectivity
* Controlled environment

---

## 🧠 Key Principle

👉 “Do not trust, do not execute — only isolate and extract”
