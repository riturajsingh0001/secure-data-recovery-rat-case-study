# ⚙️ Methodology (Technical Approach)

## 🎯 Goal

To safely recover data without executing malware or infecting another system.

---

## 🧪 Step 1: Environment Setup

* Downloaded Ubuntu ISO
* Created bootable USB using Rufus

---

## 💻 Step 2: Secure Boot

* Booted system using USB
* Selected "Try Ubuntu" (Live Mode)
* Ensured OS runs in RAM

---

## 🌐 Step 3: Isolation

* Disabled internet connection
* Prevented external communication

---

## 🔌 Step 4: Device Connection

* Connected infected Android device via USB
* Enabled MTP (File Transfer Mode)

---

## 📂 Step 5: Data Access

* Opened file manager
* Accessed device storage
* Avoided opening any files

---

## 📥 Step 6: Data Extraction

* Selected important files
* Copied to a separate clean USB drive

---

## 🔌 Step 7: Safe Removal

* Unmounted all devices
* Safely ejected USB

---

## 📴 Step 8: System Shutdown

* Powered off system
* Removed bootable USB

---

## ✅ Result

* Data recovered successfully
* No malware execution
* Host system remained secure
