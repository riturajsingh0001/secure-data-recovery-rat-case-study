# ⚠️ Threat Analysis

## 🦠 Malware Type: Remote Access Trojan (RAT)

A Remote Access Trojan (RAT) is a type of malware that allows attackers to remotely control a victim’s device without their knowledge.

---

## 🎯 Capabilities of RAT

* Remote device control
* File access and modification
* Data exfiltration
* Background execution
* Network misuse

---

## 🧨 Attack Vector

### 🔹 Social Engineering

* Malware was disguised as a normal file
* User trusted and extracted it

### 🔹 Execution Mechanism

* Extraction triggered malware activation
* No user interaction required afterward

---

## 🌐 Impact Observed

* Unauthorized data usage
* Possible remote access
* Wi-Fi bandwidth exhaustion
* Privacy compromise risk

---

## 🧠 Why Attack Was Successful

* Lack of user awareness
* No file verification
* Trust in unknown source
